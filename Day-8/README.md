# The Example 1
```java
import java.util.ArrayList;

public class Main {
    public static void main(String[] args) {
        ArrayList<String> list = new ArrayList<>();

        list.add("Java");
        list.add("C++");
        list.add("Python");

        System.out.println(list);
    }
}
```
# The Example 2
```java
import java.util.HashSet;

public class Main {
    public static void main(String[] args) {
        HashSet<String> set = new HashSet<>();

        set.add("Java");
        set.add("C++");
        set.add("Python");
        set.add("Java");

        System.out.println(set);
    }
}
```
